# TaskFlow MVP Prototype

Minimalni web prototip za pracenje:
- projekata
- taskova (TODO/In Progress/Done)
- ideja (Inbox)
- top 3 prioriteta za danas

## Pokretanje lokalno
Dovoljno je otvoriti `taskflow/index.html` u browseru.

## Napomena
Podaci se cuvaju u `localStorage` i vezani su za browser/uredjaj.
